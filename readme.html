特点

1. 友好的图形界面和使用提示

2. 一键检查更新，极速下载，无需挂梯。小版本更新还能显示Change log。

3. 支持开机自动静默更新。（强烈推荐）

该功能支持小版本自动保持最新、大版本通知更新

（点击通知即可跳转主界面，再点击检查更新即可完成大版本更新）

4. 开机更新启动后，还可以使用托盘服务，右键托盘图标即可一键检查更新。

5. 支持自动备份涂装与视角配置文件

（该功能涉及大量文件操作，可能会出现权限问题等导致无法正确备份）

6. 支持一键重装

7. 支持下载线程选择（1-8）

8. 如果觉得好用，可以点击 捐助 按钮哦～

目前软件处于beta阶段，如果出现任何问题

或还有任何功能需求，请及时向我反馈！

其他提示：

1. 如果软件出现问题，可以先删除位于

   C:/User/你的用户名/

   目录下的“.zupdater”文件夹，然后重试。

2.缓存文件位于

   C:/User/你的用户名/.zupdater/temp

   大版本更新后若不再需要压缩包文件

   可以删除temp文件夹

   （这些文件可在重装等操作时被利用，故软件不删除）

3. 若在启用了开机自动更新功能的情况下移动了软件

   那么需要进入软件重新勾选开机启动，才能试其生效！
